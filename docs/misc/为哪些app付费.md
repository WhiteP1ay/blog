# 【施工中】为哪些app付费

我们应该为软件付费，不管是操作系统、办公软件、社交媒体，还是开源软件、ai服务、生产力工具、游戏、流媒体平台。

如果市场拒绝为软件付费，软件开发者为了生存就不得不加入大量广告、付费陷阱、隐私搜集。

一个臃肿的软件是大家都不想看到的。
